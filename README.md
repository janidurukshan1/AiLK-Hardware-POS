# AiLK Hardware Shop POS System

Simple hardware shop POS system.

## Features

- Shop setup screen
- Product adding
- POS billing
- Receipt printing
- Offline local storage
- Sidebar dashboard
- Floating add button

## How To Run

1. Download project
2. Open index.html
3. Start using POS system

## Upload To GitHub

1. Create GitHub repository
2. Upload all files
3. Enable GitHub Pages

## Desktop APK/App

You can convert this into:

- Windows App using Electron
- Android APK using Capacitor
